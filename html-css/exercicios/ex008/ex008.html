Formatações

Negrito / Destaque

Tag não semantica para Negrito, letra B.

Para destaque tag Strong, essa é semantica.

Italico / Ênfase

Para Italico se utiliza a tag i (sem semantica).

Para Ênfase utilizamos a tag em (com semantica).

Marcação

se utiliza a tag Mark. e para troar a cor precisa ser em CSS.

Texto grande e pequeno

Para grande utlizamos BIG, porém o correto é usar farmatação em CSS.

E para pequeno SMALL

Texto Deletado

Utiliza-se a tag DEL

Texto adicionado

Utliza-se a tag INS

Texto sobrescrito

Utiliza-se sup

Texto subscrito

Utiliza-se SUB
